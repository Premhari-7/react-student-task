# Full Stack Bootcamp - Day 1 & Day 2 Task

## Project Overview
This project is developed as part of the Full Stack Development with AI Bootcamp. It combines Day 1 and Day 2 tasks to demonstrate the core concepts of React including components, props, state management, event handling, list rendering, and API integration.

---

## Day 1 Features

- Created reusable StudentCard component
- Used props for data passing
- Managed input using useState
- Implemented event handling
- Built a simple form with input fields
